# Proyecto Ecommerce - Frontend

Este repositorio contiene el **frontend** de la aplicación Ecommerce, desarrollado en **React + Vite**, conectado con un backend en **Node.js/Express/MongoDB** desplegado en una máquina virtual con **Docker, PM2 y Nginx**.

---

## Tecnologías utilizadas
- **React 18** con **React Router DOM** para la navegación.
- **Vite** como bundler y entorno de desarrollo.
- **Context API** para autenticación y manejo de estado global.
- **CSS responsivo** para móvil y desktop.
- **Netlify** para despliegue del frontend.
- **Backend**: Node.js, Express, MongoDB, JWT, desplegado en VM con Docker + PM2 + Nginx.
